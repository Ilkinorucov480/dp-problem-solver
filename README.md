# Dynamic Programming Problem Solver

Bu layihə 5 klassik dinamik proqramlaşdırma problemini vahid framework daxilində həll etmək üçün hazırlanır.

## Problemlər

1. 0/1 Knapsack
2. Longest Common Subsequence
3. Longest Increasing Subsequence
4. Matrix Chain Multiplication
5. Edit Distance

## Məqsəd

Layihənin əsas məqsədi memoization, tabulation və space optimization yanaşmalarını müqayisə etməkdir.

## Texnologiyalar

- Python
- Matplotlib
- Pandas
- Psutil
